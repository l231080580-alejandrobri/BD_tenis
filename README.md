BD_tenis1
Este es un sistema de una venta y gestion de tenis (interfaces de administrador/usuario).

## Funcionalidades
- Registro de usuarios (al registrarte, te pide el rol lo que hace es que ahi te registra como usuario o administrador)
- Login (solo un login para administrador/usuario, reconoce el correo si es de un usuario o administrador)
- CRUD de productos, usuarios 
- JFrame
